# QT-saver

## Project Setup
### Descripiton
Instagram saving script.

### How to install the dependencies

In your root folder, do:

  - `pip install --upgrade pip`
  - `pip install pipreqs`
  - `pip install -r requirements.txt`


### How to generate a requirements.txt file

In your root folder, do:

  - `python -m pipreqs.pipreqs . --force`
